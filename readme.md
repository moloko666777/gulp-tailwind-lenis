# Gulp Проект з TailwindCSS та Lenis

Сучасний стартовий шаблон для веб-розробки, який використовує Gulp, TailwindCSS та Lenis для плавного прокручування.

## Особливості

- 🚀 Швидка розробка з Gulp
- 🎨 TailwindCSS для стилізації
- 📜 Плавне прокручування з Lenis
- 🧩 Модульна структура з компонентами
- 🔄 Автоматичне оновлення при змінах
- 📱 Адаптивний дизайн
- 🎯 Індикатор брейкпоінтів для розробки
- 🚀 ESBuild для оптимізації JavaScript
- 🎨 Локальні шрифти Open Sans

## Структура проекту

```
├── src/
│   ├── components/     # Компоненти (head, hero, features, contact)
│   ├── css/           # SCSS файли
│   ├── js/            # JavaScript файли
│   ├── img/           # Зображення
│   ├── fonts/         # Локальні шрифти
│   └── index.html     # Головний HTML файл
├── dist/              # Зібрані файли для розробки
├── build/             # Оптимізовані файли для продакшену
└── gulpfile.js        # Конфігурація Gulp
```

## Швидкий старт

1. Встановіть залежності:
```sh
npm install
```

2. Запустіть сервер розробки:
```sh
npm run dev
```

3. Для продакшен збірки:
```sh
npm run prod
```

## Компоненти

Проект використовує модульну структуру з компонентами:

- `head.html` - мета-теги та підключення стилів
- `hero.html` - головна секція
- `features.html` - секція з особливостями
- `contact.html` - контактна форма

## Технології

- Gulp 4
- TailwindCSS 3.4
- Lenis (плавне прокручування)
- BrowserSync
- SCSS
- PostCSS
- ESBuild
- Open Sans (локальні шрифти)

## Конфігурація

Основні налаштування знаходяться у файлах:
- `config.js` - загальні налаштування проекту
- `tailwind.config.js` - конфігурація TailwindCSS
- `gulpfile.js` - завдання Gulp

## Особливості збірки

- JavaScript файли об'єднуються в один файл `main.js`
- SCSS файли компілюються в один CSS файл
- Зображення оптимізуються для продакшену
- Шрифти копіюються в папку dist/build
- Автоматичне оновлення браузера при змінах
